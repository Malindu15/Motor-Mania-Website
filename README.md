# Motor Mania - Premium Auto Marketplace

A modern, responsive, and high-performance website for buying and selling vehicles. 
Built with **HTML5, CSS3, and Vanilla JavaScript** (No Node.js backend required).

## Features
- **Modern UI**: Glassmorphism design with premium aesthetics.
- **Vehicle Listings**: detailed cards with images and specifications.
- **Search & Filter**: Keyword search, category filtering, and more.
- **Details Page**: Dedicated page for each vehicle.
- **Sell Page**: User-friendly form for submitting advertisements.

## How to Run
1. Simply double-click on `index.html` to open it in your web browser.
2. Navigate through the site using the menu links.

## Technologies Used
- HTML5
- CSS3 (Custom Properties, Grid, Flexbox)
- JavaScript (ES6+)
- Phosphor Icons (via CDN)
- Google Fonts (Outfit)

Enjoy browsing for your dream ride!
